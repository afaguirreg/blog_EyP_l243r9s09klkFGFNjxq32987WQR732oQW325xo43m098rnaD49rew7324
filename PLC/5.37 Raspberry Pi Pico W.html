Este módulo integra el dispositivo Arm® Cortex®-M0 de doble núcleo y módulo inalámbrico basado en Infineon CYW43439 para obtener una plataforma de desarrollo compatible con OpenPLC, dotada de comunicación WiFi para el desarrollo de aplicaciones IoT.

Descargar e instalar: Open PLC Editor para Windows



Iniciar OpenPLC y crear un nuevo proyecto desde el menú Archivo-->Nuevo.
