Este módulo integra el dispositivo ESP8266 para obtener una plataforma de desarrollo compatible con OpenPLC, dotada de comunicación WiFi para el desarrollo de aplicaciones IoT.

Descargar e instalar: Open PLC Editor para Windows



Iniciar OpenPLC Editor y crear un nuevo proyecto desde el menú + New Project.


Seleccionar el tipo de proyecto PLC Project.


Nombrar y asignar la ruta de almacenamiento del proyecto.


Seleccionar el lenguaje de programación y el periodo de ejecución del programa principal.


Se debe configurar el entorno OpenPLC para poder compilar y cargar los programas adecuadamente al dispositivo NodeMCU ESP8266.

- Ir al menú Device --> Configuration.

- Desmarcar la opción Compile Only.

- Seleccionar el dispositivo controlador en el menú Device y el puerto de comunicación USB en el menú Communication Port.


- Mapear los pines correspondientes a las entradas y las salidas del controlador.

- Habilitar comunicación Modbus RTU para monitoreo del controlador.


Transferir el programa al dispositivo (verificar que haya seleccionado el puerto USB correspondiente).
