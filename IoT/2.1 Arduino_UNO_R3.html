Arduino UNO R3 es un módulo integra el dispositivo Atmel AT328p para obtener una plataforma de desarrollo compatible con Arduino.

Antes de comenzar a programar se debe configurar el entorno Arduino para poder compilar y cargar los programas adecuadamente en el Arduino UNO R3 .

Abrir el menú Tools --> Board: --> Arduino AVR Boards y seleccionar la placa Arduino UNO.
