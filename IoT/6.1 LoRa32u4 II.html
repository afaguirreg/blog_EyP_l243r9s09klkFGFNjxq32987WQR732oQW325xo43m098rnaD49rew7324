Este módulo integra el dispositivo LoRa32u4 II para obtener una plataforma de desarrollo compatible con Arduino, dotada de comunicación LoRa para el desarrollo de aplicaciones IoT.

Antes de comenzar a programar se debe configurar el entorno Arduino para poder compilar y cargar los programas adecuadamente en el LoRa32u4 II.

Iniciar Arduino IDE, abrir el menú File --> Preferences....


Escribir la siguiente URL en la caja de texto Gestor de URLs Adicionales de Tarjetas: https://adafruit.github.io/arduino-board-index/package_adafruit_index.json


Abrir el menú de la izquierda BOARDS MANAGER

Buscar en la lista el gestor del módulo LoRa32u4 II.

Hacer click en el botón INSTALL.


Una vez completa la instalación, cerrar Arduino IDE y abrir nuevamente. Las librerías y ejemplos instalados estarán disponibles.

Abrir el menú Tools --> Board: Adafruit AVR Boards y seleccionar la placa Adafruit Feather 32u4.
