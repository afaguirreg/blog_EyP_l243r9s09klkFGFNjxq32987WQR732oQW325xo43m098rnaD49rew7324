NodeMCU es un módulo que integra el dispositivo ESP8266 para obtener una plataforma de desarrollo compatible con MicroPhyton, dotada de comunicación WiFi para el desarrollo de aplicaciones IoT.
El ESP8266 que incorpora NodeMCU es un SoC (System on Chip), con capacidades de 2.4 GHz Wi-Fi (802.11 b / g / n, soporte WPA / WPA2), 16 GPIO de propósito general (entrada / salida), I²C, convertidor analógico-digital (ADC de 10 bits), SPI, I²S, UART y modulación de ancho de pulso (PWM), emplea un CPU RISC de 32 bits basado en el Tensilica Xtensa LX106 funcionando a 80 MHz (o overclocked a 160 MHz). Tiene una memoria ROM de inicio de 64 KB, memoria RAM de instrucciones de 64 KB y 96 KB de RAM de datos. Memoria flash externa de 4MB (puede varíar entre diferentes versiones de modulo).

Antes de comenzar a programar se debe instalar el intérprete para poder cargar los programas adecuadamente al dispositivo NodeMCU ESP8266.

Iniciar Thonny, abrir el menú Configure interpreter...


Conectar el dispositivo NodeMCU ESP8266 a un puerto USB del PC.

Verificar que el interprete seleccionado corresponda al dispositivo a utilizar.

Seleccionar el puerto COM asignado a la placa por el sistema operativo.

Hacer click en Install or update MicroPhyton.


Ubicar y seleccionar el firmware para la placa adecuada. firmware ESP8266

Puede descargar otras versiones en: https://micropython.org/download/?mcu=esp8266

Hacer click en el botón Install.


Al finalizar la instalación del firmware el dispositivo estará listo para ser programado con MicroPhyton.
