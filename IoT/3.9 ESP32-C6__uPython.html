Este módulo integra el dispositivo ESP32 para obtener una plataforma de desarrollo compatible con MicroPhyton, dotada de comunicación WiFi para el desarrollo de aplicaciones IoT.

Antes de comenzar a programar se debe instalar el intérprete para poder cargar los programas adecuadamente al dispositivo ESP32.

Iniciar Thonny, abrir el menú Configure interpreter...


Conectar la placa ESP32 a un puerto USB del PC.

Verificar que el interprete seleccionado corresponda al dispositivo a utilizar.

Seleccionar el puerto COM asignado a la placa por el sistema operativo.

Hacer click en Install or update MicroPhyton.


Seleccionar el puerto COM asignado a la placa por el sistema operativo.

Ubicar y seleccionar el firmware para la placa adecuada. firmware ESP32

Puede descargar otras versiones en: https://micropython.org/download/esp32/

Hacer click en el botón Install.


Al finalizar la instalación del firmware el dispositivo estará listo para ser programado con MicroPhyton.
